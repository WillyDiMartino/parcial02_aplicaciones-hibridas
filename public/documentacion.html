Crea un Usuario

Para poder acceder a la API, se debe crear un usuario desde "Postman", utilizando el método POST, con el siguiente endpoint: 
http://localhost:3000/usuarios
El objeto que se debe enviar es el siguiente: 
{
name:"Nombre"
lastname:"Apellido"
username:"Usuario"
email:"Email"
password:"Contraseña"
}
Una vez creado el usuario, se podrá iniciar sesión con el mismo: 
http://localhost:3000/usuarios/login
El objeto que se debe enviar es el siguiente: 
{
email:"Email"
password:"Contraseña"
}
Se brindará un token de autenticación que tendrás que utilizarlo para poder acceder a la información.

Una vez iniciada la sesión

Las request por get te darán la siguiente información: 
http://localhost:3000/api/corredores (Devuelve la información de todos los pilotos)
http://localhost:3000/api/equipos (Devuelve la información de todos los equipos)

Corredores

Para buscar corredores hay distintas maneras: 
http://localhost:3000/api/corredores/670abe467e4829df062a3965 (Búsqueda por Id único)
http://localhost:3000/api/corredores/search/name?name=lando (Búsqueda por nombre)
http://localhost:3000/api/corredores/search/lastname?lastname=norris (Búsqueda por apellido)
Para Ordenar corredores hay distintas maneras:
http://localhost:3000/api/corredores/sort/points?order=asc (Ordena por puntos de manera ascendente)
http://localhost:3000/api/corredores/sort/points?order=desc (Ordena por puntos de manera descendiente)
Para Filtrar corredores hay distintas maneras: 
http://localhost:3000/api/corredores/filter/wins (Muestra todos los corredores que ganaron un Gran Premio)
http://localhost:3000/api/corredores/filter/championship (Muestra todos los corredores que ganaron un Campeonato)

Cada objeto tendrá estos datos:
{
        "id": "Id único del piloto",
        "name": "Nombre del piloto",
        "lastname": "Apellido del piloto",
        "team": (Dentro del objeto team se encuentra la
        información del equipo al que pertenece), {
            "id": "id único del equipo",
            "name": "Nombre del equipo",
            "base": "País de origen del equipo",
            "teamChief": "Nombre del jefe de equipo",
            "powerUnit": "Marca del motor",
            "firstEntry": Año de ingreso a la F1,
            "constructorPoints": Puntos del equipo en la temporada,
            "constructorChampionships": Campeonatos de constructores
            ganados,
            "driverOne": "Id del piloto 1",
            "driverTwo": "Id del piloto 2",
            "logoImg": "url de la imagen del logo del equipo",
            }
        "number": "Número del piloto",
        "birth": Fecha de nacimiento del piloto,
        "country": "País de origen del piloto",
        "raceWins": Carreras ganadas por el piloto,
        "podiums": Podios obtenidos por el piloto,
        "points24": Puntos obtenidos por el piloto en la temporada,
        "grandPrixEntered": Grandes premios en los que participó el piloto,
        "worldChampionships": Campeonatos mundiales ganados por el piloto,
        "driverImg": "url de la imagen del piloto",
}

Equipos

Para filtrar los equipos hay distintas maneras: 
http://localhost:3000/api/equipo/670abca17e4829df062a3963 (Búsqueda por Id único)
http://localhost:3000/api/corredores/search/name?name=mclaren (Búsqueda por nombre)
Para Ordenar equipos hay distintas maneras:
http://localhost:3000/api/equipos/sort/constructorPoints?order=asc (Ordena por puntos de equipo de manera ascendente)
http://localhost:3000/api/equipos/sort/constructorPoints?order=desc (Ordena por puntos de equipo manera descendiente)

        "constructorChampionships": Campeonatos de constructores ganados,
        "driverOne": {
            "id": "Id del piloto 1",
            "name": "Nombre del piloto 1",
            "lastname": "Apellido del piloto 1",
            "team": "Nombre del equipo",
            "number": Numero del piloto 1,
            "birth": "Fecha de nacimiento del piloto 1",
            "country": "País de origen del piloto 1",
            "raceWins": Carreras ganadas por el piloto 1,
            "podiums": Podios obtenidos por el piloto 1,
            "points24": Puntos obtenidos por el piloto 1 en la temporada,
            "grandPrixEntered": Grandes premios en los que participó el piloto 1,
            "worldChampionships": Campeonatos mundiales ganados por el piloto 1,

Administración

Para poder administrar la API, se debe iniciar sesión con el usuario y contraseña correspondiente.

Usuarios

Como administrador podrás obtener el listado de usuarios, como también crear, editar y borrar.

Para obtener el listado de usuarios:
http://localhost:3000/usuarios
También se puede buscar el usuario por su id: 
http://localhost:3000/usuarios/:id

Cada uno se crea por default con el rol:"user". En caso de querer cambiarlo, se tendrá que hacer mediante una edición de usuario.

Para editar dichos usuarios se deberá utilizar, por el metodo PUT en "Postman", el siguiente url:
http://localhost:3000/usuarios/:id
El objeto que se debe enviar es el siguiente: 
{
        name:"Nombre",
        lastname:"Apellido",
        username:"Usuario",
        email:"Email",
        password:"Contraseña",
        role:"Rol" ("admin" - "super-admin")
    }
Para borrar un usuario se deberá utilizar, por el metodo DELETE en "Postman", el siguiente url:
http://localhost:3000/usuarios/:id

Corredores

Para crear un corredor se deberá utilizar, por el metodo POST en "Postman", el siguiente url:
http://localhost:3000/api/corredores
El objeto que se debe enviar es el siguiente: 
{
        "id": "Id único del piloto",
        "name": "Nombre del piloto",
        "lastname": "Apellido del piloto",
        "team": "Id del equipo al que pertenece",
        "number": "Número del piloto",
        "birth": Fecha de nacimiento del piloto,
        "country": "País de origen del piloto",
        "raceWins": Carreras ganadas por el piloto,
        "podiums": Podios obtenidos por el piloto,
        "points24": Puntos obtenidos por el piloto en la temporada,
        "grandPrixEntered": Grandes premios en los que participó el piloto,
        "worldChampionships": Campeonatos mundiales ganados por el piloto,
        "driverImg": "url de la imagen del piloto",
    }

Si aún su equipo no está creado, se puede omitir el campo "team" y será creado con tipo de dato: null. Para luego ser editado, cuando se cree un equipo para dicho corredor.

Para editar un corredor se deberá utilizar, por el método PUT en "Postman", el siguiente url:
http://localhost:3000/api/corredores/:id

El Objeto enviado será el mismo que en la creación. Si se quiere asignarle un equipo se debe agregar a la request, luego del campo "lastname".

Para borrar un corredor se deberá utilizar, por el método DELETE en "Postman", el siguiente url:
http://localhost:3000/api/corredores/:id

Equipos

Para crear un equipo se deberá utilizar, por el método POST en "Postman", el siguiente url:
http://localhost:3000/api/equipos
El objeto que se debe enviar es el siguiente: 
{
        "id": "Id único del equipo",
        "name": "Nombre del equipo",
        "base": "País de origen del equipo",
        "teamChief": "Nombre del jefe de equipo",
        "powerUnit": "Marca del motor",
        "firstEntry": Año de ingreso a la F1,
        "constructorPoints": Puntos del equipo en la temporada,
        "constructorChampionships": Campeonatos de constructores ganados,
        "driverOne": "Id del piloto 1",
        "driverTwo": "Id del piloto 2",
        "logoImg": "url de la imagen del logo del equipo",
    }

Si se crea un equipo sin corredores, se puede omitir el campo "driverOne" y "driverTwo"y será creado con tipo de dato: null. Para luego ser editado, cuando se cree un corredor de ese equipo.

Para editar un equipo se deberá utilizar, por el método PUT en "Postman", el siguiente url:
http://localhost:3000/api/equipos/:id

El Objeto enviado será el mismo que en la creación. Si se quiere asignarle un corredor se debe agregar a la request, luego del campo "constructorChampionships".

Para borrar un equipo se deberá utilizar, por el método DELETE en "Postman", el siguiente url:
http://localhost:3000/api/equipos/:id
